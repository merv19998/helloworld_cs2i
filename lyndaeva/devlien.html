CHAPITRE 1: structure D'UN DOCUMENT HTML


HTML est un langage de description des documents utilisé sur internetpour concevoir des pages webs.

Le balisage HTML est incorporé dans le texte et est interpreté par un navigateur web.

Le XHTML quant a lui est unr évolution du HTML. Mais la plupart des informations sont valables pour les deux langages.

Une page HTML est un fichier contenant du texte et formaté avec des balises HTML.Par convention l'extension donnée aux fichiers est .htmlou.htm.Mais elle peut porter d'autre extensions:.gsp,.php.
L'ouverture d'une page HTML est effectuée a base d' un navigateur web.La creation d' une page est effectuée grace a un éditeur de texte (notepad ++;bloc note;).
